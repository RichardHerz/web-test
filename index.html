web-test ... experiments in web design for interactive learning

Experiments with embedding into web pages interactive simulations that are useable on both mobile and desktop devices.

The objective is to enhance learning by combining "reading" and "doing."

For source files, see our github project web-test. Simulations in these pages were designed using our github project lcCardLayoutToWeb. Also see our main site and Web Labs at ReactorLab.net by Richard K. Herz.

WARNING: This site may change frequently during development and may not make much sense at times. Simulations may not be operable, since we are concentrating here on page layout.

This text is in a division with the css class "class_body_text". The css file has instructions on how to lay out this division.
