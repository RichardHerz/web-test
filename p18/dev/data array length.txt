stripData is a 3D array
1st subscript is the variable (0 to numStripVars-1), 
2nd is the "x,y" point (0 to numStripPts), 
3rd is 0 (x) or 1 (y) data

so stripData[0].length returns number of "x,y" points = numStripVars+1 
so each element of stripData[0] is an array containing the x & y values for one "point"
